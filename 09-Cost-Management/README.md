# Lab 09 - Cost Management

## Objective

Learn how to monitor and optimise Azure spending.

## Topics Covered

- Cost Analysis
- Budgets
- Cost Alerts
- Resource Costs
- Forecasting

## Lab Activities

### Cost Analysis

Review resource spending.

### Budgets

Create a budget.

### Alerts

Configure spending alerts.

- Cost visibility is critical in Azure.
